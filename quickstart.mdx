Make sure you've already [installed Raffi](/installation) on your device before proceeding.

## Initial setup

<Steps>
  <Step title="Launch Raffi">
    Open the Raffi application on your desktop or mobile device. On first launch, you'll see the welcome screen.
  </Step>

  <Step title="Create an account or use local mode">
    **Desktop**: You can choose to:
    - Sign in with Ave ID (recommended for cross-device sync)
    - Continue in Local Mode (no account required, device-only storage)

    **Mobile**: Signing in with Ave ID is required for mobile devices.

    To create an Ave ID account:
    1. Click **Sign In** on the welcome screen
    2. You'll be redirected to the Ave authentication page
    3. Create your account or sign in with an existing Ave ID
    4. You'll be redirected back to Raffi
  </Step>

  <Step title="Install your first addon">
    Addons provide the streaming sources for movies and TV shows. Raffi is fully compatible with the Stremio addon ecosystem.

    1. Click the **Settings** icon (gear) in the top-right corner
    2. Navigate to **Addons**
    3. Browse the **Community Addons** section
    4. Click **Install** on any addon you want to use (e.g., Cinemeta for metadata)

    <Tip>Start with **Cinemeta** for movie and TV metadata, then add streaming addons based on your needs.</Tip>
  </Step>

  <Step title="Browse content">
    Return to the home screen to see:
    - **Popular movies and TV shows** from your installed addons
    - **Search** for specific titles using the search bar
    - **Categories** and catalogs provided by your addons
  </Step>

  <Step title="Watch your first stream">
    1. Click on any movie or TV show poster
    2. The detail page will load with information, trailers, and available streams
    3. Click the **Play** button
    4. Select a stream source from the available options
    5. Choose your preferred quality, audio track, and subtitles
    6. Enjoy your content!
  </Step>
</Steps>

## Exploring key features

<CardGroup cols={2}>
  <Card title="Continue watching" icon="clock-rotate-left">
    Your watch progress is automatically saved. Access **Continue Watching** from the home screen to resume where you left off.
  </Card>

  <Card title="Custom lists" icon="list">
    Create collections by clicking the **+** icon on any title's detail page. Organize your favorite content into custom lists.
  </Card>

  <Card title="Watch parties" icon="users">
    Watch together with friends in real-time sync. Click the **Watch Party** button when viewing content to create or join a party.
  </Card>

  <Card title="Local files" icon="folder">
    **Desktop**: Add local video files by going to Settings → Library and adding your media folders. Raffi will scan and organize your local content.
  </Card>

  <Card title="Discord integration" icon="discord">
    **Desktop**: Enable Discord Rich Presence in Settings → Activity to show what you're watching on your Discord profile.
  </Card>

  <Card title="Chromecast" icon="chromecast">
    Cast to your TV by clicking the cast icon in the video player. Raffi supports Chromecast-enabled devices.
  </Card>
</CardGroup>

## Mobile quick start

If you're using the mobile app, note these additional steps:

<Note>
The mobile app requires the streaming server component for torrent streams. The desktop app includes this server automatically. For mobile-only usage, you can:

1. Run the desktop app on your home network to provide the streaming server
2. Or deploy a standalone streaming server (see [Server documentation](/platforms/server))
</Note>

When playing torrent streams on mobile:
1. Configure the streaming server URL in the mobile app settings
2. Ensure your mobile device and server are on the same network (or configure port forwarding)
3. The mobile app will use the server for HLS transcoding

## Customizing your experience

<Steps>
  <Step title="Adjust playback settings">
    Go to Settings → Playback to configure:
    - Auto-play next episode
    - Auto-skip intros
    - Seek bar style
    - Default quality preferences
  </Step>

  <Step title="Manage addons">
    In Settings → Addons, you can:
    - Install new addons from the community
    - Configure addon settings
    - Reorder addon priority
    - Remove unused addons
  </Step>

  <Step title="Configure integrations">
    Enable optional integrations in Settings:
    - **Discord Rich Presence** (Desktop): Show your watch status
    - **Trakt**: Sync your watch history with Trakt.tv
  </Step>
</Steps>

## Troubleshooting

<Tabs>
  <Tab title="No streams available">
    If you don't see any streams for a title:
    - Make sure you have streaming addons installed (not just metadata addons like Cinemeta)
    - Try installing additional addons from different sources
    - Some content may not be available through your current addons
  </Tab>

  <Tab title="Playback issues">
    If streams fail to load or buffer excessively:
    - Try a different stream source or quality
    - Check your internet connection
    - On mobile, ensure the streaming server is running and accessible
    - Desktop: Clear cache in Settings → Storage
  </Tab>

  <Tab title="Sync not working">
    If your watch progress isn't syncing between devices:
    - Make sure you're signed in with the same Ave ID on all devices
    - Check your internet connection
    - Try signing out and back in
  </Tab>

  <Tab title="Watch party issues">
    If watch parties aren't syncing:
    - Ensure all participants have a stable internet connection
    - Only the host can control playback (play/pause/seek)
    - Try leaving and rejoining the party
  </Tab>
</Tabs>

**Having trouble?** Check the [Installation](/installation) guide for platform-specific troubleshooting, or visit the [Troubleshooting](/guides/troubleshooting) page for more help.
